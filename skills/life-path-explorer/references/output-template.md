## Usage Notes

- Adapt all section headers to user's language
- Fill every bracket with specific content from the conversation — no generic filler
- The "Evidence" fields should reference actual things the user said
- "Small experiments" must be genuinely small — completable in 1-2 weeks, low risk, but providing real signal
- The closing message should feel personal, not templated
- If the user requested the output as a file, save as markdown to their preferred location
